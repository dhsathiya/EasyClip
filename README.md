Simple and Open-Source clipboard manager. Easy to use, Light Weight and with keyboard shortcuts.

<kbd>w</kbd> : Previous clip

<kbd>s</kbd> : Next clip

<kbd>c</kbd> : Copy current clip

<kbd>q</kbd> : Minimize window
